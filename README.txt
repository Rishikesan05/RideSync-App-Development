RideSync - Real-Time Transit & Ride-Sharing Solution
===================================================

RideSync is a mobile application developed as a capstone project at the Sabaragamuwa University of Sri Lanka. It was designed to help students and staff coordinate their daily transit and ride-sharing more effectively.

Key Features:
- Real-time location tracking using Google Maps.
- Secure login and user management via Firebase.
- Simplified trip scheduling and coordination.
- Digital document and signature verification.

Technical Details:
- Framework: Flutter
- Language: Dart
- Backend: Firebase (Auth, Firestore, Storage)

Getting Started:
To run the project locally, ensure you have Flutter installed, then run:
1. flutter pub get
2. flutter run
